# BriefcaseOS Help

BriefcaseOS is a legal AI workspace for corporate transactional law.

## Main screens

- **Dashboard** — View contract stats, upload new contracts, search your portfolio.
- **Document Editor** — Review contract text. Open a contract from the dashboard table.
- **Compliance Linter** — Run compliance checks on active issues.
- **Legal Library** — Reference materials and playbooks (read in place).
- **Settings** — Profile, appearance (dark mode), and account info.

## Upload a contract

1. Go to Dashboard.
2. Click **Upload New Contract**.
3. Enter document name and client ID.
4. The contract appears in Recent Contracts. Click it to open in the editor.

## AI Assistant

Use the **?** button bottom-right to ask questions about BriefcaseOS.
With Ollama running locally (`ollama serve`, model `llama3.2`), answers use free local AI.
